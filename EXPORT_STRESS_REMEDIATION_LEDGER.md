# Export Stress Remediation Ledger

This ledger now treats exported-chat/runtime findings as first-class remediation inputs instead of prose-only notes.

## Seeded export-derived entries

- `REM_EXPORT_EXPLAIN_EXECUTION` — `explain-execution` must resolve execution-linked context or degrade with exact contract truth instead of generic missing-file failure.
- `REM_EXPORT_VALIDATE_INSPECTION_REPORT` — inspection exports and validation-complete bundles must stay distinct artifact classes.
- `REM_EXPORT_PREDICT_IMPACT_CONTRACT` — `predict-impact` must expose a live argument contract stronger than help text.
- `REM_EXPORT_CAPABILITY_TRUTH` — command support must distinguish declared, help-routable, contract-known, smoke-executed, and fully-exercised states.
- `REM_EXPORT_NAMESPACE_CONTRACT` — namespace and same-context requirements must be explicit for execution-linked artifacts.
- `REM_EXPORT_TRUST_MARGIN_ROLES` — trust parsing must distinguish margins, control roles, physical scale, and coverage-like metrics from actual error.
- `REM_EXPORT_MATH_NATIVE_CLAIMS` — claim objects must carry actual mathematical payload, not only orchestration metadata.
- `REM_EXPORT_WARM_HOST_GAP` — the warm research host remains a required complement to the cold certifier.

## Operational use

These entries should exist simultaneously as:
- persisted remediation objects under the research host
- command/artifact contract fixes where relevant
- phased items in `LINEAR_EXECUTION_RAIL.md` or in a trajectory-preserving compounding change chain applied to it

## Law

Every exported failure or friction signal should cash out into:
- a typed remediation class
- an affected-surface set
- an expected fix chain
- a status value
- and eventually a verification or deferment receipt


## Current machine absorption state

The remediation ledger is no longer only prose. It now coexists with warm-host routing, promotion-readiness, claim packets, and producer-host specs, so export-derived seams can participate in actual route choice and readiness gating rather than remaining detached notes.
